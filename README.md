# Project Name
For MSCH-C220 on 10/7/2021

Juicy Match-3

## Implementation
- Background
- SFX for grid pieces
- Coin animation
- pieces change color
- Screen shake
- Particles when selected
- Explosion when matched
- Fall tween
- Move tween
- Spin tween
- Wiggle tween w/ randomness
- Scrolling background w/ shader
